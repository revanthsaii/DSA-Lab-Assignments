3. PERMUTATIONS OF STRING

Input String: "ABC"

All Permutations:
1. ABC
2. ACB
3. BAC
4. BCA
5. CAB
6. CBA

Total Permutations: 3! = 6

Approach: Swap and backtrack

Time Complexity: O(n!)
Space Complexity: O(n)

4. COMBINATION SUM

Candidates: [2, 3, 6, 7]
Target: 7

Solutions:
1. [7]
2. [2, 2, 3]

No of Solutions: 2
Each number can be used unlimited times

Time Complexity: O(N^T/M) where T=target, M=min value
Space Complexity: O(T/M)

5. WORD SEARCH

Board:
A B C E
S F C S
A D E E

Word: "ABCCED"
Found: Yes
Path: A(0,0) -> B(0,1) -> C(0,2) -> C(1,2) -> E(2,2) -> D(2,1)

Word: "SEE"
Found: Yes
Path: S(1,0) -> E(1,3) -> E(2,3)

Word: "ABCB"
Found: No (B used twice)

Time Complexity: O(N * M * 4^L) where L=word length
Space Complexity: O(N * M)

---

Key Concepts:
- Explore all possible solutions
- Prune invalid branches early
- Restore state after recursive call
- Used for constraint satisfaction problems
